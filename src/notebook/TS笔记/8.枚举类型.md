#### 语法

```ts
enum Color {
    red,
    blue=2,
    green
}
```

#### 例子

```ts
enum Color {
    red,
    blue=2,
    green
}

console.log(Color);
// expect output
// { '0': 'red', '2': 'blue', '3': 'green', red: 0, blue: 2, green: 3 }
```
