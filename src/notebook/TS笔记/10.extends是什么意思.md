1. interface的继承

   ```ts
   interface A {}
   interface B extends A {}
   ```

2. 用在类型表达式中推断类型

   ```ts
   // 如果1是number的子类型，则N为number，否则N为unknown，可拿来做类型推断。
   type N = 1 extends number ? number : unknown;
   ```

3. 对泛型进行类型约束

   ```ts
   // T被约束成了number及其子类型
   function func<T extends number>(a: T, b: T) {
   	return a + b;
   }
   ```
